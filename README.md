# Каталог книг

## Описание
Данный проект представляет собой приложение для учёта книг, написанный полностью на PyQt5, в рамках обучения в Яндекс Лицее.

## Основные функции
+ Возможность добавлять, обновлять и удалять книгу и автора
+ Возможность добавить фотографию обложки в редакторе книг
+ Удобный поиск по каталогу с параметрами
+ Использование SQLite
+ Возможность сделать выгрузку книг в текстовый формат
